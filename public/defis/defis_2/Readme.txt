Pour intégrer le mode Nuit, créer un <div> avec comme id "BoutonModeNuit" dans votre HTML.
N'oubliez pas d'inclure le fichier javascript dans votre HTML !
Si vous souhaitez le tester sans faire de HTML, utiliser le HTML fournis dans le dossier.
Pour activer Contraste et Opacité, il faut cocher les cases correspondantes.
